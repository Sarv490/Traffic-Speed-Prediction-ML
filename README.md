# Traffic Speed Prediction Using Data Science & Machine Learning

## Overview
This project predicts traffic speed using Machine Learning and identifies congestion hotspots. It also performs route optimization using Dijkstra's Algorithm.

## Features
- Traffic Speed Prediction
- Data Cleaning & Feature Engineering
- Random Forest Regression
- Hyperparameter Tuning using RandomizedSearchCV
- Congestion Hotspot Detection
- Route Optimization
- Streamlit Deployment

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- NetworkX
- Streamlit
- Matplotlib

## Model Performance
- RMSE: 6.59
- R² Score: 0.83

## Future Scope
- Real-time traffic prediction
- Smart city integration
- GPS-based routing
